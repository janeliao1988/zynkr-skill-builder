# Traditional Chinese Filler / Disfluency Reference

Use this list as removal candidates, but keep meaning when a token is semantically necessary.

## Common fillers

- 呃
- 嗯
- 就是
- 然後
- 那個
- 你知道
- 我覺得
- 其實（when purely verbal padding)
- 這個
- 好那
- 好那個
- 對對對
- OK 好

## Repetition patterns

- same word repeated 2+ times with no added meaning
- phrase restart: "我們我們...", "這個這個..."
- abandoned starts: sentence starts then fully restarts

## STT noise to clean

- obvious homophone mistakes from context
- duplicated sentence fragments
- punctuation collapse (single very long line)

## Keep these

- domain terms: IPO, GUIDE, SOP, RAG, FE/BE/DB, HITL
- meaningful discourse signals for logic transitions
- examples and numbers used in explanation
